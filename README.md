## 📌 Overview

Foodie is a **responsive restaurant web application** built using **HTML, CSS, and JavaScript**.
It showcases a modern food website where users can explore menu items dynamically loaded from **JSON data**.

The project focuses on delivering a clean UI, smooth interactions, and a real-world restaurant browsing experience.

---

## 🚀 Features

* 🍔 Dynamic menu rendering using JSON data
* 🔍 Interactive UI for browsing food items
* 📱 Fully responsive design (Mobile, Tablet, Desktop)
* ⚡ Fast and lightweight performance
* 🎨 Clean and modern user interface
* 🔄 JavaScript-based DOM manipulation

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript (ES6)** – Logic & data handling
* **JSON** – Menu data source
